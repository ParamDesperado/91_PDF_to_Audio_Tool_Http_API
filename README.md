# 🎧 PDF to Audiobook Converter

Welcome to the **PDF to Audiobook Converter**, a premium web application designed to turn your PDF documents into structured, lifelike audiobooks in seconds.

## ✨ Features

- **Smart PDF Extraction**: Intelligently parses PDFs using layout analysis. It automatically skips tiny junk text (like page numbers and headers) and identifies large headings, creating natural pauses just like a real audiobook!
- **Math/Gibberish Detection**: Built-in algorithms analyze text density. If a PDF is full of unreadable math equations or raw code, the system will safely reject it to prevent noisy audio.
- **Premium Audio Player**: A stunning glassmorphic UI featuring custom controls (Play, Pause, Reset, Fast Forward 10s, Rewind 10s, and Playback Speed).
- **Persistent Library**: A built-in catalog automatically saves and organizes your generated audiobooks, allowing you to instantly rename, delete, or replay past documents.
